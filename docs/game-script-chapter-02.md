# 《我》第二章游戏脚本样章：绕完这一圈

版本：v0.2  
日期：2026-04-30  
基准母本：[《我》五章小说式母本](./me-story-manuscript.md)  
章节主题句：跑着跑着，世界好像在变……

## 1. 章节目标

第二章让玩家体验“重复移动让场景慢慢变化”。本章不追求找到一扇具体的门，而是让玩家在跑圈、观察粉笔痕迹、进入课桌小岛的过程中感到学校一点点松动、偏移、变得不像刚才。

玩家完成本章后，应感到：

- 学校没有突然变成异世界，而是在重复跑动中逐渐变形。
- “跑慢一点、看仔细一点”比速度更重要。
- 童年幻想像一种自然冒出来的感觉，不需要被解释成规则发明。
- 回到现实后，主角愿意给童年的小东西留一个位置。

## 4. 分镜与交互脚本

### `c02_s01_school_gate`：傍晚校门

**画面构图**

竖屏固定视角。校门在画面中部偏上，保安室小灯亮着，操场边树影露出一角。成年主角站在校门外，不进门，只沿着围墙看。画面仍是现实色彩，但校门边缘可以有一点手绘粉笔质感，为后续变化埋伏笔。

**玩家目标**

点击校门、操场影子和外墙拐角，让玩家理解主角不是要进入学校，而是想顺着外墙绕一圈。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_gate` | 点击校门 | 首次 | 文本：`校门。` 保安室灯轻闪 | 记录已调查 |
| `tap_track_shadow` | 点击操场边影子 | 任意 | 文本：`跑道在里面，看不太清。` 树影向内晃动 | 记录已调查 |
| `tap_wall_corner` | 点击外墙拐角 | 任意 | 文本：`好像可以顺着这里走一圈。` 外墙边出现细线 | 记录已调查，解锁绕行按钮 |
| `tap_security_light` | 点击保安室灯 | 任意 | 文本：`灯亮着。学校已经不是白天的学校。` | 无 |
| `start_loop` | 点击“顺着外墙走” | 已调查 3 个物件 | 主角沿墙慢慢走出画面 | `chapterProgress = entry`，进入 `c02_s02_outer_loop` |

**短旁白**

```text
他只是路过。
可脚步自己慢了一点。
```

**音频**

- 现实底噪：远处放学人声、树叶、保安室电流声。
- 音乐：几乎没有，只留很轻的木吉他泛音。
- 点击外墙时加入极轻粉笔摩擦声。

**资源占位**

- 成年主角校门站立与转身动作。
- 傍晚校门背景。
- 校门、操场影子、外墙拐角、保安室灯可点击层。
- 文本气泡 UI。

### `c02_s02_outer_loop`：外墙绕行

**画面构图**

同一段外墙反复出现，但每圈细节不同。画面下方显示一条极简跑圈轨迹，不使用传统任务条。轨迹像小孩画在作业本角落的线，玩家按住或拖动时，主角沿墙外的小路移动。

**玩家目标**

通过重复绕行感到学校逐步变松；第 4 圈必须慢拖，强调“慢下来才看得出来”。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `loop_1` | 按住轨迹前进 | `loopCount = 0` | 文本：`第一圈。学校还是学校。` 外墙保持现实 | `loopCount = 1` |
| `loop_2` | 按住轨迹前进 | `loopCount = 1` | 窗户灯光变暖一点，脚步声多一层回声 | `loopCount = 2` |
| `loop_3` | 按住轨迹前进 | `loopCount = 2` | 墙角出现粉笔灰，成年主角影子短暂变矮 | `loopCount = 3`，`chapterProgress = inner_world` |
| `loop_4_slow` | 缓慢拖动轨迹 | `loopCount = 3`，拖动速度低于阈值 | 外墙、光线和路面错开一点，画面进入操场 | `loopCount = 4`，`loopPace = slow` |
| `loop_4_fast` | 快速拖动轨迹 | `loopCount = 3`，拖动速度高于阈值 | 文本：`太快了，看不出来。` 轨迹轻轻弹回 | 不推进 |
| `tap_chalk_dust` | 点击墙角粉笔灰 | `loopCount >= 3` | 文本：`灰吹了一下，像有人刚刚跑过去。` | 记录已调查 |

**提示递进**

1. 第一次第 4 圈过快：墙边粉笔字亮一下：`第四圈要慢一点。`
2. 第二次过快：小孩影子停在轨迹上，朝玩家看一眼。
3. 第三次过快：轨迹后半段出现短暂光圈，提示慢拖区域。

**短旁白**

```text
不是跑得越快越会变。
要让学校有时间露出一点点缝。
```

**音频**

- 每圈脚步节奏略有变化：第一圈干、第二圈暖、第三圈带粉笔摩擦声。
- 第 4 圈慢拖时降低环境声，只保留鞋底和树叶。
- 成功入界时不做强切，只让校园底噪变近。

**资源占位**

- 外墙绕行背景 4 个渐变状态。
- 跑圈轨迹 UI。
- 成年主角沿墙移动、小孩影子短闪。
- 粉笔灰、窗户暖光、路面错位特效。

### `c02_s04_chalk_corridor`：粉笔走廊

**画面构图**

走廊墙面布满短箭头、圆圈、叉号和小字。镜头轻微倾斜，不做恐怖感。每条粉笔痕迹都像小孩匆忙写下的提醒，字短、笨、很认真。

**玩家目标**

收集 3 条粉笔提示，理解“慢、影子、别急着拐”这三个规则。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_chalk_hint_1` | 点击箭头 | 未收集 | 文本：`这里先别拐。` 箭头变亮 | `chalkHints += hint_no_turn` |
| `tap_chalk_hint_2` | 点击圆圈 | 未收集 | 文本：`第四圈要慢一点。` 圆圈套住一段路 | `chalkHints += hint_slow_fourth` |
| `tap_chalk_hint_3` | 点击短线 | 未收集 | 文本：`影子短的那边。` 地上影子缩短 | `chalkHints += hint_short_shadow` |
| `tap_cross_mark` | 点击叉号 | 任意 | 文本：`这里不是错，是以前试过。` | 无 |
| `enter_desk_island` | 点击走廊尽头 | `chalkHints` 数量为 3 | 走廊尽头的课桌慢慢浮起 | 进入 `c02_s05_desk_island` |

**提示递进**

1. 玩家停留 8 秒未点击：最近的粉笔灰轻轻落下。
2. 玩家只收集 1 条后点击尽头：文本：`还少一点提醒。`
3. 玩家只收集 2 条后点击尽头：未收集提示的墙面出现轻微呼吸感。

**儿童痕迹文本**

```text
这里先别拐。
第四圈要慢一点。
影子短的那边。
```

**音频**

- 走廊底噪为空教室的轻微回声。
- 每条粉笔提示被点击时加入短促粉笔点墙声。
- 三条提示收齐后，远处课桌抽屉轻响。

**资源占位**

- 粉笔走廊背景。
- 箭头、圆圈、短线、叉号可点击层。
- 儿童纸条或墙面粉笔文本 UI。
- 走廊尽头课桌浮起特效。

### `c02_s05_desk_island`：旧课桌岛

**画面构图**

课桌像岛屿一样排列，抽屉半开。桌角划痕连接成不完整的小路。不同课桌像从不同年份搬来：木纹、涂改液、贴纸痕迹、抽屉灰尘都略有差异。

**玩家目标**

阅读桌面划痕，沿划痕慢拖确认回到开头的路，并找到半截粉笔和贴纸星星。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_desk_mark_1` | 点击桌角划痕 | 未读 | 文本：`像一条没画完的小路。` 划痕轻亮 | `deskMarksRead += 1` |
| `tap_desk_mark_2` | 点击抽屉内纸条 | 未读 | 文本：`跑慢点，学校会变一点。` 抽屉停在半开 | `deskMarksRead += 1` |
| `tap_desk_surface` | 点击桌面木纹 | 任意 | 文本：`木纹像跑道旁边的小路。` | 无 |
| `collect_half_chalk` | 点击半截粉笔 | 未收集 | 获得隐藏道具：半截粉笔 | `collectedItems += half_chalk` |
| `collect_sticker_star` | 点击贴纸星星 | 未收集 | 获得隐藏道具：贴纸星星 | `collectedItems += sticker_star` |
| `confirm_slow_path` | 沿桌面划痕慢拖 | `deskMarksRead >= 2` | 学校轻轻偏移，桌面小路接回起点 | `chapterProgress = echo`，进入 `c02_s06_room_corner_echo` |

**隐藏道具**

| 道具 ID | 名称 | 获取方式 | 描述 |
| --- | --- | --- | --- |
| `half_chalk` | 半截粉笔 | 点击抽屉里的短粉笔 | `太短了，但还够画一小段。` |
| `sticker_star` | 贴纸星星 | 点击桌角旧贴纸 | `星星边缘翘起来，还是亮的。` |

**完成反馈**

桌面划痕连成一条很短的路，路并没有通向门，而是绕回一个桌角。小孩主角停下来看着那块桌角，现实房间的桌面慢慢叠上来。

**音频**

- 课桌抽屉声、木头轻响、纸条摩擦。
- 完成慢拖时，脚步声和铅笔划线声短暂合在一起。
- 回到现实前让校园环境声自然变远。

**资源占位**

- 旧课桌岛背景。
- 课桌划痕亮线状态。
- 半截粉笔、贴纸星星、抽屉纸条。
- 小孩主角停步动作。
