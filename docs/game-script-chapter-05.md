# 《我》第五章游戏脚本样章：密林之后

版本：v0.2  
日期：2026-04-30  
基准母本：[《我》五章小说式母本](./me-story-manuscript.md)  
章节主题句：前面还有啥呢？

## 1. 章节目标

第五章让玩家体验“还想往前看看”的探索感。这里不解释里世界的真相，也不把所有意象总结成答案。玩家会穿过树影，进入地下教室，遇见小时候不可能做成却一直记得的“化石”，再通过左右手小人从互相较劲变成一起往前。

玩家完成本章后，应感到：

- 结局开放在“里世界是什么”，但不开放在“主角是否发生变化”：他确实变得更愿意感受生活。
- 化石不是任务成果，而是被重新遇见的旧愿望。
- 前四章的意象不是陈列馆，而是一起把出口让出来。
- 回到现实后，主角不需要解决人生，只是多做了几件真正想做的小事。

## 4. 分镜与交互脚本

### `c05_s01_forest_edge`：旧校林边

**画面构图**

旧校后方林边。现实中没有山洞，只有树、泥土、旧墙和一条不明显的小路。旧校比记忆里矮，林子却仍然让人想知道里面还有什么。

**玩家目标**

点击旧校、树影和泥土，让探索欲从现实场景中自然出现。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_old_school` | 点击旧校 | 首次 | 文本：`旧校比记忆里矮一点。` 旧墙掉下一点灰 | 记录已调查 |
| `tap_forest_shadow` | 点击树影 | 任意 | 文本：`里面看不清，但好像还能走。` 树影向内让出缝 | 记录已调查 |
| `tap_mud` | 点击泥土 | 任意 | 文本：`泥土有一点潮。` 泥面留下浅浅脚印 | 记录已调查 |
| `tap_no_cave` | 点击林边空处 | 任意 | 文本：`现实里这里没有山洞。` | 无 |
| `enter_forest` | 点击小路 | 已调查 3 个物件 | 树影向内打开一点 | `chapterProgress = entry`，进入 `c05_s02_tree_shadow_path` |

**短旁白**

```text
知道那里没有山洞。
也还是想再走近一点。
```

**音频**

- 现实底噪：远处风声、旧校空地、鞋底踩土。
- 点击树影时树叶声靠近。
- 进入小路时现实环境声变轻，不做奇幻转场。

**资源占位**

- 成年主角林边站立。
- 旧校林边背景。
- 旧校、树影、泥土、林边空处可点击层。
- 文本气泡 UI。

### `c05_s02_tree_shadow_path`：树影小路

**画面构图**

树影铺在地上，光从叶子缝里漏下来。主角每前进一步，左右手小人的影子会短暂出现。画面仍然保留现实树林材质，入界来自影子被重新看见。

**玩家目标**

完成 3 次前进动作：拨开枝叶、跟随树影、举手做出两个小人。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `push_branch_1` | 拖开枝叶 | `forestSteps = 0` | 光斑露出，脚步向前 | `forestSteps = 1` |
| `follow_shadow_2` | 点击树影深处 | `forestSteps = 1` | 脚步声变轻，树影像小路一样连起来 | `forestSteps = 2` |
| `raise_hands_3` | 双指按住画面 | `forestSteps = 2` | 两个手影出现，先互相推一下 | `forestSteps = 3`，`chapterProgress = inner_world` |
| `tap_leaf_gap` | 点击叶缝光 | 任意 | 文本：`光不是出口，只是让你继续看。` | 无 |

**入界表现**

双指按住后，两个手影在地面短暂变成小人。树影没有把主角吞进去，而是像一条被记忆踩出来的小路，通向树洞和地下教室。

**提示递进**

1. 玩家找不到下一步：树影朝正确方向延伸一点。
2. 玩家单指点击手影区域：文本：`这次要两只手。`
3. 玩家双指按住时间不足：手影闪一下又散开，提示继续按住。

**音频**

- 树叶、泥土、轻脚步为主。
- 双指按住时加入很轻的手掌摩擦声。
- 入界时前几章的木琴、粉笔、纸声、雨声各闪过一拍。

**资源占位**

- 树影小路背景。
- 枝叶可拖层、树影路径、叶缝光。
- 左右手小人影子初始状态。
- 双指按住提示 UI。

### `c05_s03_underground_classroom`：地下教室

**画面构图**

树洞往下接到地下教室。黑板、课桌、车窗、漫画页、柜门都在，但不是陈列馆，更像走着走着碰见的旧东西。每个物件只保留前章的一个核心动作或声音。

**玩家目标**

调查前四章意象中的任意 3 个，让地下教室从“旧东西聚在一起”变成能通向化石盒的空间。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_blackboard` | 点击黑板 | 未读 | 文本：`黑板上有没擦干净的粉笔线。` 粉笔线连成短路 | `memoryObjectsRead += blackboard` |
| `tap_window_memory` | 点击车窗 | 未读 | 文本：`车窗里还有一点灯光。` 远处车灯亮一下 | `memoryObjectsRead += window` |
| `tap_manga_page` | 点击漫画页 | 未读 | 文本：`纸边轻轻翘着。` 纸片朋友影子跑过 | `memoryObjectsRead += manga` |
| `tap_cabinet_memory` | 点击柜门 | 未读 | 文本：`里面很黑，但不是可怕的黑。` 雨声变近又变稳 | `memoryObjectsRead += cabinet` |
| `collect_glass_marble` | 点击课桌抽屉里的弹珠 | 未收集 | 获得隐藏道具：玻璃弹珠 | `collectedItems += glass_marble` |
| `enter_fossil_table` | 点击靠里的泥台 | 已调查任意 3 个记忆物件 | 泥台上的透明盒亮起 | 进入 `c05_s04_impossible_fossil` |

**隐藏道具**

| 道具 ID | 名称 | 获取方式 | 描述 |
| --- | --- | --- | --- |
| `glass_marble` | 玻璃弹珠 | 点击课桌抽屉里的弹珠 | `一颗小小的玻璃弹珠，把灰色折出一点颜色。` |

**完成反馈**

被调查过的物件不会消失，而是各自让出一条小缝。车窗灯光、粉笔线、漫画页和柜门黑暗一起指向靠里的泥台。

**音频**

- 地下教室使用混合环境：粉笔、车声、纸页、雨声都很轻。
- 每调查一个记忆物件，对应前章音色响一下。
- 点击泥台时，所有音色退后，只留下手碰到透明盒的轻声。

**资源占位**

- 地下教室背景。
- 黑板、车窗、漫画页、柜门、课桌抽屉、泥台可点击层。
- 玻璃弹珠道具。
- 记忆物件亮起状态。

### `c05_s04_impossible_fossil`：不可能的化石

**画面构图**

教室靠里的泥台上放着一个透明盒子，盒盖有点旧，边角沾着干掉的土。里面躺着一块小小的“化石”，像泥里压住了一只很轻的甲虫影子。旁边有旧科普书页和一小块潮湿泥土。这里的重点是“遇见”，不是挖掘。

**玩家目标**

调查化石盒、旧科普书和泥土，理解这个愿望没有成功过，但曾经非常认真。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_fossil_box` | 点击透明盒 | 未读 | 文本：`不像真的，但它在这里。` 盒子轻轻反光 | `fossilCluesRead += box` |
| `tap_book_why` | 点击旧科普书页 | 未读 | 文本：`图很好看，所以想要一个。` 书页上图案亮一下 | `fossilCluesRead += book` |
| `tap_old_mud` | 点击盒边干土 | 未读 | 文本：`等了很久，泥土没有变魔法。` 土屑落下一点 | `fossilCluesRead += mud` |
| `collect_impossible_fossil` | 点击化石盒 | 已读 3 条线索，未收集 | 获得隐藏道具：不可能的化石 | `collectedItems += impossible_fossil` |
| `enter_finger_stage` | 点击墙上手影 | 已收集化石 | 两个手影开始互相推挤 | 进入 `c05_s05_finger_stage` |

**隐藏道具**

| 道具 ID | 名称 | 获取方式 | 描述 |
| --- | --- | --- | --- |
| `impossible_fossil` | 不可能的化石 | 读完化石盒、科普书、泥土线索后点击透明盒 | `没成功，但很认真。` |

**完成反馈**

主角没有把化石从泥里挖出来，而是把透明盒捧起来。墙上的两个手影因此安静一点，从互相较劲变成试着一起护住盒子。

**音频**

- 化石场景非常安静，只保留泥土、纸页和透明盒轻响。
- 点击旧科普书时加入很短的翻书声。
- 收集化石盒时不做奖励音，只让心跳和呼吸轻轻同步。

**资源占位**

- 泥台化石场景。
- 透明化石盒、旧科普书页、干土可点击层。
- 化石盒反光特效。
- 收集提示 UI。

### `c05_s05_finger_stage`：手指小人舞台

**画面构图**

墙面成为影子舞台。左右手小人开始互相推挤，随后通过玩家双指拖动完成合作。背景里能隐约看到黑板、车窗、课桌、柜门和漫画页。

**玩家目标**

完成 3 个合作动作：分开不互撞、一起抬起木板、一起护住化石盒。动作完成后，前四章意象各自让路，晨光出现。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `separate_hands` | 双指向两侧拖开 | 初始 | 两个小人不再互相撞，墙面稳定 | `handCoopActions += separate` |
| `lift_board` | 双指同时上滑 | 已完成分开 | 小人一起抬起想象木板，课桌挪出空隙 | `handCoopActions += lift` |
| `hold_fossil` | 双指向中间合拢 | 已完成抬起 | 小人一起护住化石盒，车窗灯光连成路 | `handCoopActions += hold` |
| `tap_wrong_single_hand` | 单指拖动 | 任意 | 文本：`一个人会推偏。` 手影轻轻弹回 | 无 |
| `finish_hand_stage` | 点击晨光 | 完成 3 个动作 | 两个手影短暂重合后一起消失 | `chapterProgress = echo`，进入 `c05_s06_morning_echo` |

**提示递进**

1. 玩家只用单指：文本：`这次要一起。`
2. 双指方向不对：其中一个手影看向另一个。
3. 停留 8 秒：墙面出现淡淡动作影子，提示下一步方向。

**完成反馈**

动作一变，整间地下教室跟着松动。黑板上的字自己排开，车窗里的灯汇成一条路，课桌挪出空隙，柜门不再只通向躲藏，而通向外面更亮一点的地方。

**音频**

- 手影动作使用轻打击和手掌摩擦声。
- 每完成一个动作，前章对应音色加入一拍。
- 晨光出现时所有声音变成普通早晨环境声。

**资源占位**

- 手指小人影子舞台。
- 三个合作动作影子状态。
- 晨光出口。
- 双指手势提示 UI。

### `c05_s06_morning_echo`：普通晨光

**画面构图**

现实房间晨光。桌上放着新买的笔、半截粉笔、旧书签、透明塑料尺、玻璃弹珠和化石盒。手机在一旁，窗外有普通树影。主角还是成年人，但站姿和表情比第一章松一些。

**玩家目标**

复访最终桌面物件，完成三个现实小行动：看窗外、整理桌面、给朋友发消息。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_new_pen` | 点击新买的笔 | 任意 | 文本：`这支笔是自己选的。` | `echoObjectsVisited += pen` |
| `tap_fossil_echo` | 点击化石盒 | 任意 | 文本：`不是真的化石，但是真的很想要过。` | `echoObjectsVisited += fossil` |
| `tap_marble_echo` | 点击玻璃弹珠 | 已收集玻璃弹珠 | 文本：`灰色里也能折出一点颜色。` | `echoObjectsVisited += marble` |
| `tap_window_morning` | 点击窗户 | 任意 | 文本：`外面只是早晨，但好像还可以出去看看。` | `echoObjectsVisited += window` |
| `arrange_table_items` | 轻拖任意收集物到桌面一排 | 已复访至少 2 个物件 | 桌面物件摆齐，不需要完美对齐 | `echoObjectsVisited += arranged_items` |
| `send_friend_message` | 点击手机消息 | 已复访至少 3 个物件 | 文本：`最近怎么样？` 手机发送气泡停在画面上 | 章节结束 |

**结束条件**

复访任意 3 个物件，整理一次桌面，并发送朋友消息后，结尾短句出现；点击短句后游戏结束或返回章节选择。

**结尾短句**

```text
前面还有啥呢？
```

**状态变化**

- `chapterProgress = complete`
- `perceptionStage = ending_echo`
- `chapterUnlocks.05_after_forest = complete`
- `visitedEchoes += c05_morning_echo`

**音频**

- 现实房间晨光环境声，加入非常轻的前四章音色残影。
- 点击玻璃弹珠时有清亮但很小的玻璃声。
- 发送消息后不做胜利音乐，只让窗外树叶声变清楚。

**资源占位**

- 晨光房间背景。
- 成年主角自然站姿或坐姿。
- 新买的笔、粉笔、旧书签、透明尺、玻璃弹珠、化石盒、手机、窗户可点击层。
- 最终桌面收集展示 UI。
