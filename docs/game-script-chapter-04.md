# 《我》第四章游戏脚本样章：柜子里的呼吸

版本：v0.2  
日期：2026-04-30  
基准母本：[《我》五章小说式母本](./me-story-manuscript.md)  
章节主题句：那个让我安心的角落去哪了？

## 1. 章节目标

第四章让玩家体验“躲起来时又刺激又安心”的矛盾感。柜子不是逃避失败的象征，而是小时候会让人心跳加快、又觉得稳当的小空间。本章核心交互围绕柜门、呼吸声、脚步声、雨声和身体缩进去的感觉展开。

玩家完成本章后，应感到：

- “先躲一下”不是失败，而是一种具体的自我照顾方式。
- 躲藏的趣味来自又怕被发现、又知道自己有位置可待的矛盾感。
- 声音摆放是建立安静边界，不是抽象 UI 谜题。
- 回到现实后，主角愿意给自己一小段不用解释的安静时间。

## 4. 分镜与交互脚本

### `c04_s01_work_pressure`：工位夜晚

**画面构图**

办公室工位，屏幕消息浮着但不刺眼。成年主角坐在椅子上，肩膀略紧。周围人声被处理成模糊背景，桌面抽屉像一个小小的暗处。

**玩家目标**

点击消息、抽屉和顶灯，理解主角想藏起来不是因为大灾难，而是因为世界太需要他立刻回应。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_messages` | 点击消息 | 首次 | 文本：`消息还在跳。` 通知声叠一层 | 记录已调查 |
| `tap_desk_drawer` | 点击抽屉 | 任意 | 文本：`抽屉里很暗。` 抽屉开一条缝 | 记录已调查 |
| `tap_overhead_light` | 点击顶灯 | 任意 | 文本：`灯有点太亮。` 顶灯轻微闪白 | 记录已调查 |
| `tap_chair_gap` | 点击椅子下方 | 任意 | 文本：`这里放不下人。` | 无 |
| `close_laptop` | 点击合上电脑 | 已调查 3 个物件 | 声音突然变小，抽屉缝变成柜门缝 | `chapterProgress = entry`，进入 `c04_s02_cabinet_doors` |

**短旁白**

```text
他不是想消失。
只是想先不要被任何声音碰到。
```

**音频**

- 现实底噪：办公室空调、键盘、人声、手机提示音。
- 点击消息时提示音略密，点击抽屉时声音短暂变闷。
- 合上电脑后留一个很小的静默点。

**资源占位**

- 成年主角工位坐姿。
- 工位夜晚背景。
- 消息、抽屉、顶灯、椅子下方可点击层。
- 文本气泡 UI。

### `c04_s02_cabinet_doors`：柜门开合

**画面构图**

一排旧储物柜叠在办公室抽屉缝上。每次开合后，现实办公室色彩退后，旧校气味和雨声靠近。柜门不是恐怖物件，而像一个不确定但可进入的小空间。

**玩家目标**

通过 3 次开合让玩家感到“心怦一下”的入口动作。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `open_cabinet_1` | 点击柜门 | `cabinetOpenCount = 0` | 文本：`里面黑黑的。` 柜内出现木头和铁皮味 | `cabinetOpenCount = 1` |
| `close_cabinet_1` | 点击关门 | `cabinetOpenCount = 1` | 心跳声轻响，外界声音变闷 | `cabinetOpenCount = 2` |
| `open_close_repeat` | 长按柜门开合 | `cabinetOpenCount = 2` | 雨声靠近，柜门缝里出现旧教室 | `cabinetOpenCount = 3`，`chapterProgress = inner_world` |
| `tap_lock_slot` | 点击柜门锁孔 | 任意 | 文本：`锁孔很小，像在偷看。` | 无 |

**入界表现**

第三次开合后，画面不是跳到教室，而是从柜门缝里先听到雨声，再看到窗边。小孩主角坐在柜子里，膝盖缩起来。

**音频**

- 柜门开合声从现实抽屉声逐渐变成旧储物柜金属声。
- 每次关门都让外界声音短暂变闷。
- 第三次长按时加入雨点打窗框声。

**资源占位**

- 旧储物柜背景。
- 柜门开/关/半开状态。
- 小孩主角柜内剪影。
- 柜门缝光特效。

### `c04_s03_rain_classroom`：老教室雨声

**画面构图**

雨天旧教室。窗边雨声、门外脚步声、主角呼吸声以可拖动的小声波图标出现，但图标应像贴在场景里的小纸片，而不是纯 UI 面板。

**玩家目标**

完成一次声音摆放：雨声放窗边，脚步声放门外，呼吸声藏进书包。这里负责教会本章的声音逻辑。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `drag_rain_to_window` | 拖雨声到窗边 | 任意 | 雨声变厚，窗框亮一点 | `soundSlots.rain = window` |
| `drag_steps_to_door` | 拖脚步声到门外 | 任意 | 门外脚步远一点，走廊暗一点 | `soundSlots.steps = door` |
| `drag_breath_to_bag` | 拖呼吸声到书包 | 任意 | 呼吸声变轻，书包鼓起一点 | `soundSlots.breath = bag` |
| `tap_wrong_sound` | 放错声音 | 任意 | 文本：`这里太响了。` 声音纸片退回 | 无 |
| `finish_sound_place` | 点击安静下来的课桌 | 三个声音都正确 | 教室变成可穿过的安静空间 | 进入 `c04_s04_cabinet_islands` |

**提示递进**

1. 第一次放错：对应目标位置轻轻发声，例如窗边雨声更明显。
2. 第二次放错：小孩主角看向正确方向。
3. 第三次放错：声音纸片边缘出现短暂光圈。

**儿童痕迹文本**

```text
雨放窗户。
脚步放外面。
呼吸藏起来。
```

**音频**

- 声音摆放前，雨、脚步、呼吸互相抢占空间。
- 每放对一个声音，混音层次更清楚。
- 全部放对后不是绝对安静，而是“终于不用马上回应”的安静。

**资源占位**

- 雨天旧教室背景。
- 雨声、脚步声、呼吸声纸片图标。
- 窗边、门外、书包声音槽位。
- 小孩主角教室待机。

### `c04_s04_cabinet_islands`：储物柜群岛

**画面构图**

不同柜子通向不同小空间：课本柜、树叶柜、空柜。每个柜子都同时有一点紧张和安心。镜头更靠近身体，让玩家注意肩膀、膝盖、背部贴住柜壁的感觉。

**玩家目标**

试过 3 个柜子，理解“最合适的地方”不等于最大或最安全，而是紧张和安心刚好并存。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `enter_book_cabinet` | 点击课本柜 | 未试过 | 文本：`书挤着肩膀，反而很稳。` 画面边缘变窄 | `cabinetChecked += book` |
| `enter_leaf_cabinet` | 点击树叶柜 | 未试过 | 文本：`里面有雨后树叶的味道。` 雨声变软 | `cabinetChecked += leaf` |
| `enter_empty_cabinet` | 点击空柜 | 未试过 | 文本：`空空的，心跳声很清楚。` 心跳声靠近 | `cabinetChecked += empty` |
| `tap_too_big_cabinet` | 点击很大的柜子 | 任意 | 文本：`太空了，反而藏不住。` | 无 |
| `collect_plastic_ruler` | 点击透明尺 | 未收集 | 获得隐藏道具：透明塑料尺 | `collectedItems += plastic_ruler` |
| `enter_forest_edge` | 点击树影柜门 | `cabinetChecked` 数量为 3 | 树影从柜门里探进走廊 | 进入 `c04_s05_forest_edge_hide` |

**隐藏道具**

| 道具 ID | 名称 | 获取方式 | 描述 |
| --- | --- | --- | --- |
| `plastic_ruler` | 透明塑料尺 | 点击树叶柜里的透明尺 | `透明、安静，像一条可以藏起来的边。` |

**音频**

- 每个柜子有不同的近距离声音：书页、树叶、空柜心跳。
- 点击大柜子时回声稍空，提示它不是合适躲藏点。
- 试完三柜后，雨声与树影声合在一起，引出下一场景。

**资源占位**

- 储物柜群岛背景。
- 课本柜、树叶柜、空柜、大柜子、树影柜门。
- 透明塑料尺道具。
- 柜内近景遮罩。

### `c04_s05_forest_edge_hide`：山林边缘躲猫猫

**画面构图**

树影从窗外延伸到走廊，像一场温柔的躲猫猫。没有怪物，没有倒计时。这里不再重复声音摆放，而是让玩家调整身体进入小空间：缩进去、留门缝、慢呼吸。

**玩家目标**

完成一次躲藏姿态，让“刺激又安心”的身体感成立。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `drag_child_inside` | 向内拖动小孩主角 | 初始 | 小孩缩进柜子，画面边缘贴近 | `hidePosture.inside = true` |
| `adjust_door_gap` | 上下拖动柜门缝 | 已缩进去 | 门缝留下一条光，外面树影不再刺眼 | `hidePosture.gap = narrow` |
| `hold_breath_slow` | 长按 3 秒后松开 | 门缝已调整 | 呼吸声从急变慢，心跳不再抢出来 | `hidePosture.breath = slow` |
| `tap_tree_shadow` | 点击树影 | 任意 | 文本：`它不是来抓你，只是在外面晃。` | 无 |
| `finish_hiding` | 点击半开的柜门 | 三个姿态条件完成 | 小孩主角缩进去又探头，走廊安静下来 | `chapterProgress = echo`，进入现实回声 |

**提示递进**

1. 未缩进去就调门缝：文本：`先把自己放进去。`
2. 门缝留太大：文本：`太亮了，心会一直跳。`
3. 长按时间太短：文本：`再慢一点，呼吸会自己回来。`

**完成反馈**

柜门半开着，里面留着一小块黑暗。小孩主角探头看向走廊，外面什么大事都没发生，但声音终于不再追着他。

**音频**

- 山林边缘保留雨声，但混入树叶声。
- 长按呼吸时，呼吸声跟随按压逐渐放慢。
- 完成时环境声保留，不进入完全静音。

**资源占位**

- 山林边缘走廊背景。
- 小孩主角缩进柜子、探头动作。
- 可调门缝 UI 或柜门可拖层。
- 呼吸长按反馈特效。

### `c04_s06_quiet_room_echo`：安静房间

**画面构图**

现实房间夜晚。灯调暗，手机朝下，桌上有透明塑料尺。第二章桌角和第三章旧书签可以在画面边缘出现，提示这些变化都留在现实里。

**玩家目标**

关闭提醒、调暗灯、长按椅子坐下，完成“给自己留 10 分钟”的现实回声。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_phone_mute` | 点击手机 | 任意 | 文本：`消息还在，但可以晚一点。` 手机翻面，提示音停止 | `echoObjectsVisited += phone` |
| `tap_lamp_dim` | 点击台灯 | 任意 | 文本：`光线低一点，房间没有坏掉。` 光线变软 | `echoObjectsVisited += lamp` |
| `tap_plastic_ruler_echo` | 点击透明尺 | 任意 | 文本：`尺子透明得像一小截安静。` | `echoObjectsVisited += plastic_ruler` |
| `sit_quietly` | 长按椅子 3 秒 | 已复访至少 2 个物件 | 主角安静坐下，画面停留 2 秒 | 章节结束 |

**结束条件**

复访任意 2 个物件并完成长按坐下后，结尾短句出现；点击短句后章节结束。

**结尾短句**

```text
先躲一下，也很好。
```

**状态变化**

- `chapterProgress = complete`
- `perceptionStage = chapter_04_echo`
- `chapterUnlocks.05_after_forest = true`
- `visitedEchoes += c04_quiet_room_echo`

**音频**

- 手机静音后，现实房间底噪变清楚。
- 台灯调暗时加入很轻的电流声。
- 坐下完成后保留一段不尴尬的安静。

**资源占位**

- 安静房间夜晚背景。
- 手机翻面、台灯变暗、椅子长按层。
- 透明塑料尺回声道具。
- 成年主角坐下待机动作。
