# 《我》第三章游戏脚本样章：纸片朋友

版本：v0.2  
日期：2026-04-30  
基准母本：[《我》五章小说式母本](./me-story-manuscript.md)  
章节主题句：那时候以为，他们会永远陪着我。

## 1. 章节目标

第三章让玩家体验小时候对纸片朋友的喜欢。重点不是创作新故事，也不是画技成长，而是“看完很喜欢，于是想照着画下来”的冲动。里世界中的伙伴来自主角扭扭歪歪的临摹稿，温暖、笨拙、不准确，但很亲。

玩家完成本章后，应感到：

- 临摹不是考试，也不是证明画得好，而是把喜欢多留一会儿。
- 纸片朋友不能出现真实 IP 名字、标志和高相似组合，只保留“小时候努力画过”的温度。
- 分镜重排不是页码题，而是按主角当年喜欢的情绪顺序重读一次。
- 回到现实后，主角愿意承认自己现在也可以继续喜欢漫画。

## 4. 分镜与交互脚本

### `c03_s01_bookshelf`：旧书架

**画面构图**

现实房间。书架偏暗，旧漫画从一排书中露出卷边。桌上有铅笔、橡皮屑和旧练习纸。第二章留下的粉笔和贴纸星星在桌角边缘可见，但不是本章主交互。

**玩家目标**

点击旧漫画、铅笔和旧纸，重新建立“看完以后想画下来”的动作链。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_old_manga` | 点击旧漫画 | 首次 | 文本：`封面软了，边角卷起来。` 旧漫画从书架滑出一点 | 记录已调查 |
| `tap_pencil` | 点击铅笔 | 任意 | 文本：`铅笔头有点钝。` 铅笔轻轻滚动 | 记录已调查 |
| `tap_old_paper` | 点击旧纸 | 任意 | 文本：`纸背面还有淡淡的线。` 临摹纸半透明亮起 | 记录已调查，解锁翻页 |
| `tap_bookshelf_gap` | 点击书架缝隙 | 任意 | 文本：`以前喜欢的东西，没有完全塞回去。` | 无 |
| `open_manga` | 点击打开漫画 | 已调查 3 个物件 | 纸页翻开，临摹纸盖到漫画页上 | `chapterProgress = entry`，进入 `c03_s02_page_tracing` |

**短旁白**

```text
他不是突然想画得更好。
只是又想把喜欢留下来一点。
```

**音频**

- 现实底噪：房间静音、书页摩擦、远处车声。
- 点击漫画使用干燥纸页声，点击铅笔使用木头轻滚声。
- 音乐使用很轻的拨弦，避免怀旧过满。

**资源占位**

- 现实书架背景。
- 成年主角翻书动作。
- 旧漫画、铅笔、旧纸、书架缝隙可点击层。
- 文本气泡 UI。

### `c03_s02_page_tracing`：翻页与临摹

**画面构图**

漫画页占画面中央，半透明临摹纸覆盖其上。虚线不是评分线，而是旧铅笔痕迹。玩家沿着几条抖动的线慢慢描，线条歪一点也会保留，形成主角小时候不准确但认真的笔触。

**玩家目标**

完成 3 段宽容描线，让临摹稿从纸上站起来。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `trace_face` | 沿脸部虚线拖动 | 未完成 | 线条歪一点也算完成，纸上角色眨一下 | `traceLinesCompleted += 1` |
| `trace_hand` | 沿手部虚线拖动 | 未完成 | 手被描得有点大，角色轻轻挥手 | `traceLinesCompleted += 1` |
| `trace_bag` | 沿书包虚线拖动 | 未完成 | 尾线抖动，临摹稿从纸上立起来 | `traceLinesCompleted += 1` |
| `tap_bad_line` | 点击画歪的地方 | 任意 | 文本：`画歪了，但看得出来很喜欢。` | 无 |
| `flip_page` | 轻扫翻页 | `traceLinesCompleted < 3` | 文本：`还想把这一页多留一会儿。` | 无 |

**入界表现**

第三段描线完成后，临摹纸边缘立起来，书桌变成纸页地面。成年主角不直接消失，而是他的手先变小，再带出小孩主角。

**提示递进**

1. 玩家描线过快：虚线轻轻变粗，文本：`慢一点也没关系。`
2. 玩家偏离虚线很远：线条自动吸回一小段，不扣分。
3. 玩家停留 8 秒：铅笔尖轻轻指向下一段虚线。

**音频**

- 铅笔描线声随拖动速度变化，但不做失败音。
- 第三段完成时加入轻木琴三音。
- 纸页立起时使用柔和纸张折叠声。

**资源占位**

- 漫画页与半透明临摹纸背景。
- 三段虚线描线层。
- 临摹稿站起动画。
- 铅笔尖提示光效。

**资源占位**

- 收集提示 UI。

### `c03_s04_panel_corridor`：分镜走廊

**画面构图**

四格分镜悬在空中。格子上没有页码，只有画面线索：第一格是主角盯着角色脸看，第二格是手停在下一页前，第三格是临摹纸被举起来，第四格是朋友把书签递过来。每格都有一条可点击的情绪线索。

**玩家目标**

通过观察格子内的动作、声音和纸边文字，推导情绪顺序：喜欢、期待、得意、告别。

**分镜线索**

| 分镜 ID | 情绪 | 视觉线索 | 点击线索文本 |
| --- | --- | --- | --- |
| `panel_like` | 喜欢 | 角色脸被描了很多遍，眼睛旁有反复擦过的痕迹 | `这张脸最想先留下来。` |
| `panel_wait` | 期待 | 手停在下一页角，纸页还没翻过去 | `下一页还没看，但已经想看了。` |
| `panel_proud` | 得意 | 临摹纸被举到光下，线很歪但很亮 | `画得不像，也想举起来看看。` |
| `panel_bye` | 告别 | 朋友站在舞台边递出书签 | `这一格不是丢掉，是下次还能找到。` |

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_panel_clue_like` | 点击喜欢格线索 | 未读 | 显示线索文本，格子边缘变暖 | `panelCluesRead += like` |
| `tap_panel_clue_wait` | 点击期待格线索 | 未读 | 显示线索文本，翻页声停半拍 | `panelCluesRead += wait` |
| `tap_panel_clue_proud` | 点击得意格线索 | 未读 | 显示线索文本，临摹纸亮一下 | `panelCluesRead += proud` |
| `tap_panel_clue_bye` | 点击告别格线索 | 未读 | 显示线索文本，书签轻晃 | `panelCluesRead += bye` |
| `drag_panel_like` | 拖动喜欢格 | 任意 | 放入顺序槽 | 更新 `panelOrder` |
| `drag_panel_wait` | 拖动期待格 | 任意 | 放入顺序槽 | 更新 `panelOrder` |
| `drag_panel_proud` | 拖动得意格 | 任意 | 放入顺序槽 | 更新 `panelOrder` |
| `drag_panel_bye` | 拖动告别格 | 任意 | 放入顺序槽 | 更新 `panelOrder` |
| `confirm_panel_order` | 点击确认 | 四格已放入 | 顺序正确则走廊亮起，错误则格子轻轻散开 | 正确时解锁剧场 |

**正确顺序**

```text
喜欢 -> 期待 -> 得意 -> 告别
```

**提示递进**

1. 第一次排序错误：文本：`不是页码，是心里先后亮起来的地方。`
2. 第二次排序错误：未读线索的分镜边角自动翘起。
3. 第三次排序错误：纸片朋友依次看向正确的第一格和第二格。

**完成反馈**

四格按情绪顺序连起来后，不恢复成原漫画，而是变成一条纸片舞台的路。临摹朋友从格子边缘跑向舞台。

**音频**

- 每个情绪格有轻微不同音色：喜欢为木琴、期待为翻页、得意为铅笔敲桌、告别为纸张合上。
- 排序错误只使用柔和纸页散开声。
- 排序正确时四种音色按顺序短暂连成旋律。

**资源占位**

- 分镜走廊背景。
- 四格分镜与顺序槽 UI。
- 情绪线索点击状态。
- 排序成功连接特效。

### `c03_s05_paper_theater`：纸片剧场

**画面构图**

纸箱和便利贴搭成小舞台。临摹朋友要走到舞台尽头，但舞台缺一块桥板。旧书签从舞台侧边露出，既是桥板，也是最后送给主角的东西。

**玩家目标**

拖动旧书签修补舞台，看纸片朋友完成一场短短的小演出，并获得旧书签。

**交互**

| 交互 ID | 输入 | 触发条件 | 反馈 | 状态变化 |
| --- | --- | --- | --- | --- |
| `tap_paper_friend` | 点击朋友 | 任意 | 文本：`它没有画准，但它在等你。` | 无 |
| `tap_stage_gap` | 点击舞台缺口 | 首次 | 文本：`这里少一小块路。` | 记录已调查 |
| `drag_bookmark_bridge` | 拖动旧书签到缺口 | 已调查缺口 | 书签变成桥，纸片朋友通过 | `theaterFixed = true` |
| `tap_friend_bow` | 点击谢幕朋友 | `theaterFixed = true` | 文本：`你不需要画得很像，才能再找到我们。` | 解锁旧书签 |
| `collect_old_bookmark` | 点击旧书签 | `theaterFixed = true`，未收集 | 获得隐藏道具：旧书签 | `collectedItems += old_bookmark` |

**隐藏道具**

| 道具 ID | 名称 | 获取方式 | 描述 |
| --- | --- | --- | --- |
| `old_bookmark` | 旧书签 | 修好纸片舞台后点击书签 | `夹过很多页，也等过很久。` |

**完成反馈**

纸片朋友走过书签桥，停在舞台边回头。它们没有留下来陪主角永远玩，而是把旧书签交给他，像给下一次打开书留一个位置。

**音频**

- 纸箱舞台使用轻轻的纸板碰撞声。
- 书签放下时使用短促翻页声。
- 谢幕时音乐只停一小拍，不做煽情铺满。

**资源占位**

- 纸片剧场背景。
- 纸箱舞台、缺口、便利贴云。
- 旧书签桥状态。
- 临摹朋友走路与谢幕小动画。

- 新书桌面回声版背景。
- 新漫画、旧书签、铅笔、书店小票可点击层。
- 成年主角翻书待机动作。

## 7. 灰盒验证重点

第三章进入灰盒时，优先验证以下问题：

- 描线交互是否宽容，不让玩家以为是绘画评分。
- 临摹朋友是否足够温暖，同时不会像真实 IP 复刻。
- 分镜重排是否能靠线索推导，而不是靠猜正确答案。
- 纸片剧场是否能让玩家愿意停一下看它们。
- 章节总时长是否控制在 8-12 分钟。
